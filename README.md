# link-share
Keep your links in one place and share them with your friends.

### Required env parameters setup

on Mac or Linux:
```
export LINK_SHARE_DB_SUPERUSER=link-share-usr
```

on Windows
```
C:\Users\you\data-access> set LINK_SHARE_DB_SUPERUSER=link-share-usr
```

## Run the app
In order to run the app simply run the command bellow.

```
docker compose up --build
```
